实验器材:
	STM32F103C8T6---48脚LQFP封装-128Kflash-工业级（-40-85度）
	
实验目的:
	实现指纹模块的基本功能
	
硬件资源:
	0.不允许使用以下端口：
		PA0---控制冰箱门；
		PA1---门状态获取；
		PA9、PA10---USART调试用的串口；
		PA4、PA5、PA6、PA7---SPI1和网口通信；
	1.PB12---LED指示灯
	2.PA2---TX2--白线
	  PA3---RX2--黄线
	3.PB0---指压信号输入---

	
实验现象:
	1.如果有未录入的指纹按下，led灯亮起。led灭了之后，表示指纹读取完毕，提示放手，再按压一次，led灯亮起，led灭了之后，表示指纹读取完毕。开冰箱门。
	2.如果有已录入的指纹按下，led灯亮起，led灭了之后，表示指纹读取完毕。一会之后led灭，开门。
